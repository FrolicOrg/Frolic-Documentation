# Getting Started

Follow the steps below to setup Frolic for your dashboard.

import { Steps } from 'nextra/components'
 
<Steps>

### Clone Frolic

```
git clone https://github.com/frolicorg/frolic.git
```
 
### Connect your Database
 
Connect your database to Frolic by entering your database configurations in [.env](https://github.com/frolicorg/frolic/blob/prod/app/server/.env) file. Read more about connecting your database to Frolic [here](/getting-started/app-configurations#database).

### Run Project
 
Use docker to run Frolic

```
docker-compose up --build
```

### Query your Data
 
Send `POST` request to Frolic `/api` to query data from database. Refer to sample request below.

```
curl --location 'http://localhost/api' \
--header 'Content-Type: application/json' \
--data '{
      "metrics": [
        {
          "field": "products.price",
          "aggregate_operator": "sum"
        }
      ],
      "dimensions": [
        {
          "field": "products.category"
        }
      ]
}
'
```

Checkout [API documentation](/api-reference) to learn more about API parameters.

</Steps>
